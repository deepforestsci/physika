Physika
=====================

Physika is designed to simplify the representation of equations governing diverse physical systems,
and the computational methods used to approximate their solutions. Physika is a type-based differentiable
language program, that checks for dimension and data type correctenss.

.. code-block:: text

   example.phyk → Lexer → Parser → AST → Type Checker → Runtime (PyTorch)
